1. Executive Summary

CentrixPlus proposes an AI-powered enhancement to Al Mulla Exchange's call center — designed to ensure every customer call is answered, whether by an intelligent AI voicebot or a live agent, 24 hours a day, 7 days a week.

The solution is fully integrated with the existing Centrix Dial, Centrix PBX, and CRM infrastructure. Unlike cloud-only platforms that require data to leave Kuwait, our approach keeps all data on-premise, delivers tighter integration, and costs significantly less than third-party alternatives.

AI Voicebot

Answers inbound calls instantly — handles rate inquiries, transaction status, FAQ, and complaints. Supports Arabic and English. Provides 24/7 coverage including after-hours.

Live Agent Transfer

When the AI cannot resolve, the call transfers seamlessly to a live agent. The agent sees the full AI conversation, customer details, and sentiment — no need for the customer to repeat.

Ticketing & Escalation

If an agent cannot resolve on the call, a ticket is auto-created with AI summary, assigned to the right team, and tracked with SLA monitoring. Customer gets SMS confirmation.

Sentiment Analysis

Every agent call is analyzed automatically — sentiment, resolution status, compliance, and agent performance. Management dashboard with real-time alerts for critical calls.

2.3 How Calls Are Divided Between AI and Human Agents

The system intelligently routes each call to the right handler — AI bot or human agent — based on the nature of the inquiry:

Call TypeHandled ByExpected %What Happens
Rate inquiriesAI Bot~25%Bot fetches live rates from the system and responds instantly
Transaction statusAI Bot~20%Bot looks up reference number and provides status
General FAQ / Branch infoAI Bot~10%Bot answers from knowledge base
After-hours callsAI Bot~10%Bot handles fully — logs tickets for next business day if needed
Complaints / IssuesAI Bot → Agent~15%Bot captures details, creates ticket, transfers to agent with context
Complex / VIP / ReturningHuman Agent~20%Call goes directly to agent queue
How it works in practice: Every call is first answered by the AI bot. If the bot detects the customer needs a human agent — either because they request one, or the issue is complex — the call is transferred instantly. The human agent receives a screen pop showing the full AI conversation, so the customer never has to repeat themselves.

6.2 Data Privacy & Security

  • All call recordings remain on-premise in Kuwait — never uploaded to external storage
  • Only live audio streams are sent to cloud AI services during active bot calls — encrypted (TLS 1.3)
  • Cloud AI providers do not store or train on API data
  • All analysis results stored locally in the on-premise database
  • Access secured via Cloudflare Tunnel + role-based authentication
  • No customer data leaves Kuwait permanently

6.3 Language Support

Full support for Arabic and English. The AI voicebot uses Google and Azure speech services with proven Gulf Arabic accuracy for both speech recognition and voice synthesis. Arabic Neural TTS provides natural-sounding Arabic responses. Additional languages (Hindi, Urdu, Tagalog) can be added based on customer demographics.
